move(DIRECTION)

현재 위치에서 주어진 값대로 한 칸 이동합니다.
move(RIGHT); move(LEFT); move(UP); move(DOWN);처럼 쓰면 됩니다.
주위 미로의 상태를 보고 프로그램이 판단해서 호출하는 식으로 씁시다.

findNear(arr)

현재 위치의 미로 정보를 3x3 배열로 받아옵니다.
int arr[3][3]; findNear(arr);으로 사용하면 arr에 미로의 정보가 담겨옵니다.
0은 벽, 1은 길을 의미합니다.

getSuccess()

목적지에 도착했으면 1, 도착하지 못했으면 0을 반환합니다.
while 루프등을 사용할 경우 각 단계마다 이 함수를 호출해서 도착했으면 함수를 종료해주세요!
while(1)
{
        move(RIGHT);
        if( getSuccess()==1 )
                return;
}
